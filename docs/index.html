Run InsureIQ Locally
Live in under 5 minutes
1
Clone the Repository
git clone https://github.com/Momna-bit/insureiq.git cd insureiq
2
Install Dependencies
pip install -r requirements.txt
3
Set Up API Keys (.env file)
ANTHROPIC_API_KEY=your_claude_key OPENAI_API_KEY=your_openai_key SNOWFLAKE_ACCOUNT=your_account SNOWFLAKE_USER=your_username SNOWFLAKE_PASSWORD=your_password SNOWFLAKE_WAREHOUSE=COMPUTE_WH SNOWFLAKE_DATABASE=INSUREIQ_RAW SNOWFLAKE_SCHEMA=CLAIMS
4
Train Model + Index Claims
python src/train_model.py python src/generate_historical.py python src/index_historical.py
5
Start the API
python src/api.py
6
Launch the Dashboard
streamlit run src/dashboard.py
7
Open the Dashboard
🛡️ Open InsureIQ → localhost:8501

The Problem

Insurance fraud is a
$80 billion annual crisis.

Traditional fraud detection relies on manual review — slow, inconsistent, impossible to scale. InsureIQ was built to change that with AI.

$80B

Lost to insurance fraud every year

That's roughly 10% of all insurance payouts globally. Manual review catches less than 20% of fraudulent claims. The rest slip through — costing insurers billions every year.

🧠
The AI Brain
Claude AI reads every claim the way a senior analyst would — extracting claimant details, incident type, amount, and anything that doesn't add up. No forms, no templates. Pure intelligence applied to raw text.
🔍
The Memory Layer
Every claim is instantly compared against 500+ historical cases stored as mathematical patterns in ChromaDB — like giving the AI years of institutional experience in milliseconds.
⚡
The Risk Engine
An XGBoost ML model scores each claim's fraud probability in real time. High risk triggers investigation. Low risk moves to fast approval. Clear, consistent decisions every time.
📊
The Executive Dashboard
Every claim, score, and red flag flows into Snowflake's three-layer data warehouse (RAW → TRANSFORM → MART) and surfaces in a custom-built HTML executive dashboard — showing fraud rates, risk distribution, and claim volume in real time.
